HTML Fundamentals

What is HTML?

HTML stands for Hypertext Markup Language. It is interpreted by the browser, and it is used to create the structure of web pages. Each element you see on a page (each paragraph, header, etc.) is there because of HTML (and, yes, the formal name for these components of a web page is "element").

What is the basic structure of an HTML document?

An HTML document has a standard skeleton that consists of several opening and closing tags. These tags include:

  1. A Doctype Declaration: This tells the browser what type of document it is interpreting.
  2. A Root Tag: The root tag of any HTML document is the "html" tag.
  3. A Head Tag: This tag contains metadata about our document. This metadata includes information about the set of characters being used and the page's title.
  4. A Body Tag: The body tag contains other tags which represent the elements on the page you can see. Here you'll find paragraph tags, div tags, etc.

How are HTML elements rendered?

Remember that HTML is an interpreted language. This means that your browser knows how to make sense of HTML -- no compilation needed. When your browser is interpreting HTML, it renders each element on the page sequentially. And this isn't just true of HTML elements -- it's also true of any script or style tags you might be using.

There are three different ways to style your web pages. You can use:

  1. Inline CSS
  2. Style Tags (Internal CSS)
  3. External Style Sheets

Inline CSS entails styling your elements right here in the HTML file using the "style" attribute (hence the term "inline"). This paragraph, for instance, is being styled using inline CSS.

What are some of the ways of selecting HTML elements?

Knowing how to select elements is important for multiple reasons, two of which are the following:

  1. You need to select HTML elements in order to apply CSS styles to them.
  2. You need to select HTML elements in order to manipulate your web page using JavaScript (we'll talk more about this later).

Here are just a few of the ways in which we can select HTML elements in CSS

  • by id
  • by class
  • by element
  • by attribute

As you've already seen, we can assign an ID, class, or attribute to an element of our choosing. Note, however, that you should only use a specific ID for one element while classes are traditionally used to group many elements together.
